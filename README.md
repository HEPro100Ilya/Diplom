# Программный комплекс обнаружения сетевых атак и автоматического реагирования

###  Описание проекта
Разработка системы класса IDS/IPS (Intrusion Detection/Prevention System) на базе Suricata. 
Комплекс предназначен для мониторинга сетевого трафика, выявления подозрительной активности и автоматической блокировки атакующих узлов на уровне фаервола (Netfilter/iptables).

### Технологический стек
*   **ОС:** Debian 12 (Headless)
*   **Детектор:** Suricata IDS
*   **Блокировщик:** Netfilter (iptables) + Python Manager
*   **Backend:** Python 3 (FastAPI)
*   **БД:** SQLite
*   **Уведомления:** Telegram Bot API
*   **Frontend:** HTML5/JS (Tailwind CSS)

###  Функциональные возможности
1.  Анализ трафика в реальном времени.
2.  Автоматический парсинг логов (eve.json).
3.  Блокировка IP-адресов на основе Severity (уровня угрозы).
4.  Личный кабинет администратора (Dashboard).
5.  Мгновенные алерты в Telegram.
